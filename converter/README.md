# First Java Project: Converter

This is my first Java project. I started it while watching a YouTube tutorial:

```text
https://www.youtube.com/watch?v=KXZfh19JN-A&t=1197s
```

In the video, "Comment Coder" introduces a unit converter project. Instead of following the tutorial to the end, I decided to stop there and build the project entirely on my own.

## Features

- Convert kilometers to miles
- Convert Celsius to Fahrenheit
- Convert kilograms to pounds
- Interactive menu
- Input validation for invalid choices
- Ability to perform multiple conversions without restarting the program

## Technologies

- Java
- Scanner
- Switch statements
- While loops

## How to Run

Compile the program:

```bash
javac converter.java
```

Run the program:

```bash
java converter
```

## Example

```text
Choose your converter:
1. Km to miles
2. °C to °F
3. Kg to LB
4. Quit

1
Please, type your value in km:
10
Result: 6.21371 miles
```
